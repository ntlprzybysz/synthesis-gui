About This Project

This website provides a user-friendly graphical interface (GUI) for speech synthesis tools developed by Stefan Taubert. The aim of this project is to offer an alternative way of accessing these tools to those who are unable or prefer not to use the original command-line interface (CLI).

The primary tool featured in this GUI is a CLI designed for training Tacotron 2 using .wav and .TextGrid pairs. You can explore the repository for Tacotron on GitHub to learn more. The second tool is a CLI WaveGlow that is used for training WaveGlow with .wav files. See Waveglow on Github. You can listen to an audio sample of a synthesized text: "The North Wind and the Sun were disputing which was the stronger, when a traveler came along wrapped in a warm cloak" (headphones recommended).

The web interface for these tools has been developed by Natalia Przybysz. It incorporates technologies such as Django, Bootstrap, Gunicorn, Nginx, RabbitMQ, Celery and Celery Beat and it is written in JavaScript, HTML, CSS and Python. The project's source code is available on GitHub.

If you are intrested in contributing to this project, citing it or finding out more about it, please reach out to its owner.
